# Telegraphum
Transliterate text to morse and play as sound or flashlight.

## Features
* Transliterate
  * Text to morse
  * Morse to text
* Play
  * Morse
* Flash morse as light
* Copy the result as text

## Availablity
This application is available on the iOS [App Store](app2.it/telegraphum).
